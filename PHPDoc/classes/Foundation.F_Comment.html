get_By_Photo()

get_By_Photo(integer  $photo_ID, boolean  $order_DESC = FALSE) : array

Rethrives the comments posted on a photo passing its ID.

The output will be in ASCendent order = from the first comment made to the latest one.

Parameters

integer $photo_ID

The photo's ID selected to get the comments from
boolean $order_DESC

Whether to order result in DESCendent order. Default: ASCendent

Throws

\Foundation\queries

In case of connection errors

Returns

array —

The comments made for the photo. How to access the array:

  • "id" => the comment's ID
  • "text" => the text of the comment
  • "user" => the user's username that wrote the comment
