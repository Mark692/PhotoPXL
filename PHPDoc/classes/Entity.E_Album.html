get_By_User()

get_By_User(string  $username, integer  $page_toView = 1, boolean  $order_DESC = FALSE) : array

Rethrives the album IDs, Titles and Thumbnails of a user by passing its username.

Parameters

string $username

The user's username selected to get the albums from
integer $page_toView

The page number to view. It influences the offset
boolean $order_DESC

Whether to order result in DESCendent order. Default: ASCendent

Throws

\Entity\queries

In case of connection errors

Returns

array —

The user's albums (IDs, Titles, Thumbnails) and the total of albums created. How to access to the array:

  • "id" => the album's ID
  • "title" => the album's title
  • "cover" => its cover
  • "type" => its type
  • "tot_album" => The number of albums matching the query
